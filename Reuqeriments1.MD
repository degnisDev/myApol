Basado en los conocimientos adquiridos, crearemos un sitio web Front-end bajo las siguientes especificaciones:
1.
Encuentre el próximo dispositivo inteligente que quiere comprar o haga una lista de los mejores dispositivos o marcas, creando un sitio web sobre dispositivos inteligentes.
2.
En la página inicial puede tener un listado de los últimos celulares (móviles) o portátiles organizados por fecha de lanzamientos, marcas, tipos, agregar filtros básicos y un buscar.
3.
Al dar clic en un dispositivo del listado, se debe mostrar el detalle del aparato; aquí puede agregar la reseña, sinopsis, actores, fotos, o cualquier otra información relevante que necesite agregar. Estos dos ejemplos pueden servirle de apoyo:
o
https://www.exito.com/tecnologia/computadores-y-accesorios/computadores-portatiles
o
https://www.ktronix.com/computadores-tablet/computadores-portatiles/c/BI_104_KTRON
o
Nota: recuerde que solo se esperan ver los datos solicitados en esta actividad.
4.
Otra funcionalidad que puede agregar es la de implementar un sistema de comentario, donde los usuarios puedan dar su opinión sobre un determinado dispositivo.
5.
Por último, debe tener un sistema de administración para incorporar los datos base para la consulta anteriormente solicitada. Para esto debe tener el diseño de una base de datos.
6.
Puede utilizar la herramienta ATOM o el IDE Visual Studio Code para la gestión de su código.
7.
Recuerde utilizar buenas prácticas de maquetación de código XHTML o usar un Framework CSS como Bootstrap, Angular o React.
8.
Debe implementar la solución una arquitectura por capas, basada en eventos o una arquitectura de microservicios para, posteriormente, implementar un Back-end completo.
